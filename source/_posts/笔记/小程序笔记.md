1：引入外部css
```
@import "../ship/ship.wxss";
```
2：无法使用背景图片，需要使用线上资源。

3：保存数据
```
this.setData({
   num : 2
})
```

4：尽量避免在小地方用ID名称 因为id可以随着数据的 e.currentTarget.id
或者直接在标签里面写 data-id = "xx" 原型链（结构树）中会出现一个datatest的对象


5：block操作属性元素之间没办法嵌套 block

6：获取链接中id参数
```
onload:function(options){
this.setData({
      channel_id: options.id
    })
}
```
